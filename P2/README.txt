Trabajo en windows, así que para ejecutarlo con la consola de UBUNTU, hago SHIFT + click derecho dentro de
la carpeta del proyecto y le doy a "Abrir shell de linux aquí". Entonces escribo en la consola
"python3 main.py".

Sale un menú con opciones para poder ejecutar todos los ejercicios por el orden que se quiera.

Para que funcione todo, a parte del vídeo BBB necesito el vídeo cutBBB que es el primero minuto del vídeo
original. Este lo uso en el ejercicio 3 para que el resize no tome tanto tiempo como con el vídeo entero.
En el aula global he subido un zip que contiene los 2 vídeos, antes de ejecutar el código hay que
descargarlos y meterlos en la carpeta del main.py.
